Tabbed

A tablike jQuery plugin

This tabbed function is used to quickly build highly-extensable tab boxes for displaying related, but distinct, areas of content in a modern way. Can work with any markup which contains a set of "link"-elements with relations to a set of "block"-elements.

Example

  • Green Pants
  • Blue Jeans
  • Brown Chinos
  • Black Tights
